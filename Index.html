En este sitio tratare de describir algunas de las herramientas de Devops, como y que podemos alcanzar aplicando este comcepto

¿Que es Devops?

Sobre Devops podemos encontrar multiples definiciones en la web, desde que es una metodologia, una cultura hasta que es el uso de herramientas para automatizar procesos, lo cierto es que sea cual sea la definicion que tomemos podemos decir que Devops busca la integracion de dos areas, desarrollo e infraestructura.

¿Cuales herramientas se sugieren?

  1. Pipeline
  2. Git
  3. Terraform
  4. Microsoft Azure
  5. Ansible

¿Por que deberia aprender Devops?

  • Por que nos permite automatizar procesos
  • Por que nos permite realizar mejoras continuas
  • Por que nos ofrece alta disponibilidad
  • Los despliegues de infraestructura bastante rapidos con IaC
